# toy_text_stats

텍스트 파일 하나를 읽어서 기본 통계 정보를 출력하는 프로그램입니다.

## Features

1. txt파일 읽기
2. 각 함수의 용도에 맞게 txt파일 처리
3. 총 줄 수, 총 단어 수, 총 글자 수, 가장 긴 단어, 평균 단어 길이 계산
4. 각 요소들 값 출력

## Tech Stack

1. python3
2. Standard Library(os, string)

## Project Structure

toy_text_stats/

 -text_stats.py

 -sample.txt

## How to Run

python text_stats.py

## Example Output

총 줄 수: 11

총 단어 수: 59

총 글자 수: 430

가장 긴 단어: transportation.

평균 단어 길이: 6.254237288135593
